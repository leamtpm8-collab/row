Zugangsdaten für den Login generieren

Diese Seite läuft komplett lokal in deinem Browser — nichts wird irgendwohin gesendet. Öffne sie einfach direkt als Datei (Doppelklick / Drag & Drop in den Browser), gib dein gewünschtes Passwort ein, und kopiere die beiden erzeugten Werte in die Vercel-Umgebungsvariablen AUTH_PASSWORD_HASH und AUTH_SECRET.

AUTH_PASSWORD_HASH
AUTH_SECRET
In Vercel: Project → Settings → Environment Variables → beide Werte eintragen (Production + Preview) → neu deployen. Danach diese Datei am besten lokal löschen oder das Passwortfeld einfach leer lassen.
